# Documentation

This directory contains the formal written report associated with the
project.

The document provides a detailed explanation of:
- The dataset and exploratory analysis
- The mathematical formulation of PCA implemented from scratch
- Interpretation of principal components and loadings
- Construction of a continuous PDF and CDF using numerical integration
- Polynomial extrapolation and discussion of limitations

The report complements the notebooks by providing theoretical background,
derivations, and detailed discussion of results.
