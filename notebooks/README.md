# Notebooks

This directory contains the Jupyter notebook used to develop the
multivariate analysis presented in this project.

## Contents
- Exploratory Data Analysis (EDA) and correlation analysis
- Principal Component Analysis implemented from scratch
- Interpretation of principal components using loadings
- Continuous PDF and CDF estimation for PC1 using numerical integration
- Polynomial extrapolation for scenario analysis

## Usage
Notebooks are designed to be executed from top to bottom to reproduce
all results.
